# 📒 Contact Book

A simple Contact Book application built using Python. This project allows users to store and manage contact information through a command-line interface. It is designed to help beginners understand how to use dictionaries, loops, and conditional statements in Python.

## ✨ Features

- ➕ Add a new contact
- 📋 View all contacts
- 🔍 Search for a contact by name
- ❌ Delete a contact
- 🚪 Exit the application

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code

## 📚 Concepts Used

- Dictionaries
- Variables
- User Input (`input()`)
- `while` Loop
- `if-elif-else` Statements
- Dictionary Operations

## ▶️ How to Run

1. Install Python on your system.
2. Open the project in Visual Studio Code.
3. Run the following command:

```bash
python contact_book.py
```
